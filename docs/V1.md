# AntiTrend – V1 Definition

## 1. Problem

V1 reduces exposure to fast fashion and ultra-fast commerce sites so users can shop in alignment with their values.

## 2. Target User

Online shoppers who want to avoid fast fashion and fast commerce.

## 3. Scope (What V1 Blocks)

Sites that fall under three categories:
- Fast Fashion
- Ultra-Fast Fashion
- Fast Commerce

## 4. Explicitly Not in V1

AntiTrend does not:
- Judge users  
- Track behavior  
- Push ideology  

## 5. First 60-Second Experience

1. User selects which categories to block.  
2. User activates or downloads the block list.  
3. User begins browsing with those sites filtered.
